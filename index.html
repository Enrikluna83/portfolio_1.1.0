Acerca de Mí

Mi enfoque es aprender y aplicar nuevas tecnologías para resolver problemas reales, siempre buscando mejorar y optimizar cada proyecto en el que trabajo. Aunque soy relativamente nuevo en la industria, me encanta el desafío de crear aplicaciones completas, desde la interfaz de usuario hasta la lógica de servidor, asegurando una experiencia de usuario fluida y un código limpio y mantenible.

Habilidades

Construyo interfaces de usuario interactivas utilizando tecnologías como HTML, CSS, JavaScript y frameworks modernos como React. Me apasiona crear experiencias visualmente atractivas y funcionales, asegurándome de que la interfaz sea intuitiva y accesible.

HTML

Soy capaz de crear páginas web desde cero utilizando la estructura estándar de HTML5. Creación de estructura contenida, enlaces y navegación, integración de imágenes, formularios, tablas, etiquetas semánticas, uso de comentario, entre otras habilidades.

CSS3

Reglas de CSS, seleccionadores como clases, IDs, y elementos y cómo aplicar estilos a los elementos HTML. Familiaridad con las propiedades esenciales de CSS como color, fuentes, background y justificación. Moldeo flex box y grid para calcular el tamaño y el espacio de los elementos animaciones y transiciones.

Javascript

Manejo de sintaxis: Variables, tipos de datos, operadores, estructuras de control (condicionales, bucles). Funciones: Declaración de funciones, parámetros, retorno de valores y funciones flecha. Scope: Conocimiento de los diferentes tipos de ámbito (global, local) y cómo funcionan `let`, `const` y `var`..
